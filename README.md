# calculator-fedoseeva-kya
Программа представляет собой калькулятор для выражений, содержащая поддержку операторов +,-,*,/,(,). Инструкция к программе включает шаги по ее запуску и использованию.

Шаги по запуску программы:
 1)Скопируйте код программы в текстовый редактор.
 2)Сохраните файл с расширением .c (например, main.c).
 3)Откройте командную строку или терминал.
 4)Перейдите в директорию, где находится файл main.c.
 5)Компилируйте программу с помощью команды gcc -o main.c [main.c](main.c).
 6)Запустите программу с помощью команды ./main.

Использование программы:
 1)Введите математическое выражение в инфиксной форме, используя стандартные операторы сложения (+), вычитания (-), умножения (*) и деления (/), а также круглые скобки.
 2)Нажмите Enter, чтобы получить результат.
